About the Bot

Unlimited ForceSub Bot allows you to manage unlimited channels and force subscriptions for your Telegram bots with an easy plug-and-play setup. Built using Pyrogram and MongoDB, it ensures intelligent user approvals and high performance.

Features

  • Unlimited forced subscription channels supported.
  • Auto-approve users sending join requests.
  • Easy multi-bot setup using setup.json.
  • Fast MongoDB-based database operations.
  • Protect against fake joins with smart checks.

Setup Instructions

Basic setup involves:

1. Clone the repository:
   git clone https://github.com/yourusername/filestorebot.git

2. Edit setup.json with your bot token, channel IDs, API ID, and API hash.

3. Run the bot:
   python3 main.py

Full instructions available in the README.md.
